
# ITCAD Department / IT電腦輔助設計部

The ITCAD (Information Technology Computer-Aided Design) department is responsible for developing and maintaining CAD software solutions and IT infrastructure that supports our engineering and design operations.

ITCAD（資訊技術電腦輔助設計）部門負責開發和維護CAD軟體解決方案以及支持我們工程和設計運營的IT基礎設施。

## Key Responsibilities / 主要職責

- CAD software development and customization / CAD軟體開發與客製化
- IT infrastructure management / IT基礎設施管理
- Database design and maintenance / 資料庫設計與維護
- Software integration and automation / 軟體整合與自動化
- Technical support for design tools / 設計工具技術支援
- System security and backup management / 系統安全與備份管理
